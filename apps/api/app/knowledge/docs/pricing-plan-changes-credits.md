# Plan Changes, Downgrades, and Credits

Plan downgrades and credits can reduce booked value without indicating an outage. They should be handled as commercial context unless support tickets or product events show a technical trigger.

## Analyst Notes

- Credits need the credit memo ID before they are cited.
- Downgrade claims should include the old plan, new plan, effective date, and approver.
- Do not infer a pricing problem from MRR drop alone when failed invoices explain the delta.
