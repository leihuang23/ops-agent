# Support Macro - CSV Import Instability

Use this draft when admins report intermittent CSV import failures and product events show `import_failed` for affected accounts.

## Draft

Hi {{admin_name}}, we are investigating intermittent CSV import failures affecting recent workspace updates. The current workaround is to split files over 20,000 rows into smaller batches and retry after five minutes.

## Evidence Needed

Include the import job ID, account ID, failure timestamp, and ticket ID before requesting approval.
