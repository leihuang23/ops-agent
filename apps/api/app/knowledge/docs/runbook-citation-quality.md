# Citation Quality Runbook

Citation quality is a product requirement. A final report is not acceptable when it makes unsupported claims, cites unretrieved documents, or hides failed tool calls.

## Citation Rules

- Metric claims cite SQL result summaries.
- Support claims cite ticket IDs.
- Product claims cite event names and account counts.
- Procedure claims cite knowledge source IDs and chunk IDs.
- Uncertainty should be stated when evidence is incomplete or conflicting.
