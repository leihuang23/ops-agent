# Incident Response - Report Export Filters

The report export filter bug creates duplicate support tickets and high frustration, but it is not usually the direct cause of a paid MRR drop.

## Signals

- Product events include repeated `report_export` activity.
- Tickets mention missing filters, incorrect CSV files, or duplicate scheduled exports.
- High-priority tickets may cluster in Product Support.

## Response

Deduplicate tickets, publish a workaround, and do not classify as billing unless invoice evidence also exists.
