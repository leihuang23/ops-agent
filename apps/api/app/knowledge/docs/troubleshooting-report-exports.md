# Troubleshooting Report Exports

Report export tickets often mention missing filters, unexpected rows, or duplicate scheduled emails. These issues should be cited from tickets and product events, not from invoice data.

## Workaround

Ask admins to open the report page, apply filters manually, and use direct export. Scheduled exports should be paused for affected saved views until the filter fix is deployed.
