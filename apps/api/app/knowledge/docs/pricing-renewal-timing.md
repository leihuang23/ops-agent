# Renewal Timing and Invoice Recognition

Paid invoice MRR is recognized on the invoice date after the invoice is paid. A renewal invoice dated late in a weekly window can create timing noise, but it should not create a persistent drop when renewals are collected.

## Recognition Rules

- Paid invoices count toward paid invoice MRR for the invoice date window.
- Failed invoices are excluded until recovered.
- Void invoices are excluded and should be explained by cancellation or contract correction.
- Manual recovery should preserve the original invoice ID in the evidence bundle.
